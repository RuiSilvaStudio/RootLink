# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| main    | ✅ Active development |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.
Instead, send an email to **rui.fc.silva@proton.me** with:

- A brief description of the issue
- Steps to reproduce (if applicable)
- Affected versions

You should receive a response within 72 hours. If you do not, please follow up.

## Scope

This policy covers the RootLink codebase hosted at
https://github.com/RuiSilvaStudio/RootLink. Third-party dependencies should
be reported to their respective maintainers.
